# Foundations Resources

## Git & Version Control

- [Pro Git Book](https://git-scm.com/book/en/v2) - Scott Chacon & Ben Straub - Comprehensive guide covering Git internals and advanced workflows
- [Git from the Bottom Up](https://jwiegley.github.io/git-from-the-bottom-up/) - John Wiegley - Understanding Git's graph structure and object model
- [A Visual Git Reference](https://marklodato.github.io/visual-git-guide/index-en.html) - Mark Lodato - Visual guide to core Git commands and their effects

## Probability & Statistics

- [Probability Cheatsheet](https://github.com/wzchen/probability_cheatsheet) - William Chen - Comprehensive probability reference covering distributions, theorems, and key concepts
